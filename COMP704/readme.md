## Build instructions for make
Assuming that the following is available:
* latex (texlive-scheme-full or windows equlivient)
* latexmk (should be part of the latex package)
* qpdf - tool for optimising/repairing PDFs

I use make to generate optimised PDFs:

```bash
$ make clean
$ make
```
